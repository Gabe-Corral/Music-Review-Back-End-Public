# Music Review App

This is the back-end porstion of a music review web app that I built mostly for fun.

**Technologies Used:**

- Ruby version: 2.6.1

- Rails Verison: 6.0.3 

- PostgreSQL

- Spotify API

**Setup:**

```
git clone https://github.com/Gabe-Corral/Music-Review-Back-End.git
cd Music-Review-Back-End
bundle install
rails db:create
rails db:migrate
rails s
```
